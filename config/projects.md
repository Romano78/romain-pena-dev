## 1. Atari

**URL:** https://atari.com
**Type:** Full Storefront Build
**Industry:** Gaming / E-commerce
**Image:** `/public/Atari-Logo.png`

### What I did
Full pixel-perfect implementation of the Atari storefront from Figma — built before the Dawn framework existed, so every component was custom-coded from scratch. On top of the design build, added several advanced features: MetaMask wallet sync so users could buy NFTs directly on the site, countdown timers, a country and tag-based product showcase system, and a custom fulfillment backend that detects Trevco-tagged orders and routes them to the right shipping partner for custom merchandise production.

### Key deliverables
- Pixel-perfect Figma implementation
- Custom component library (pre-Dawn framework)
- MetaMask wallet sync for NFT purchases
- Country & tag-based product showcase logic
- Custom Trevco fulfillment backend
- Countdown timers

### SEO angles
- Shopify custom storefront development
- NFT + e-commerce integration
- Custom Shopify Liquid development
- Pre-Dawn Shopify theme architecture
- Custom fulfillment automation Shopify

## 4. Animo etc

**URL:** https://animoetc.com
**Type:** Custom App & Backend
**Industry:** Pet Retail (Quebec / Canada)
**Image:** TBD

### What I did
Backend-focused engagement for Animo etc, a Quebec-based pet supply chain. Built a custom app that syncs their internal product database with Shopify — automatically creating and updating products on a schedule. Also engineered a store location selector feature: users pick their store, and the storefront shows the correct inventory for that location. The challenge was that Shopify natively deducts inventory from the closest warehouse location — we built a custom backend that intercepts this and reroutes deduction to the correct store location instead. Added full English/French bilingual support throughout.

### Key deliverables
- Custom database → Shopify product sync app
- Store location selector (UI + backend)
- Custom inventory deduction logic (override Shopify native behavior)
- English / French bilingual support

### SEO angles
- Custom Shopify app development
- Shopify inventory location management
- Shopify multi-location inventory
- Product sync Shopify custom app
- Shopify bilingual store French English
- Quebec Shopify developer
